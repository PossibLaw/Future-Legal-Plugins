# possiblaw

Umbrella plugin that exposes canonical PossibLaw command entrypoints under a single `/possiblaw:*` namespace:

- `/possiblaw:build-plugin`
- `/possiblaw:guardrails`
- `/possiblaw:legal`
- `/possiblaw:vibe-coding`

Install:

```bash
claude plugin install possiblaw --marketplace PossibLaw
```
